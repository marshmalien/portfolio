NannyDash

Final Project for The Iron Yard

I worked with a team of 4 developers to create this project in two weeks. NannyDash is a dashboard application that simplifies communication and job requests between agencies, families, and nannies. I implemented many of the dashboard views using Angular 1.5's component method and used AngularJS routing and templating to keep the page loads snappy. The project was built with AngularJS front-end and a Rails back-end.
